# Content Rating questionnaire — answer key

Play uses the IARC questionnaire — same one Apple, Microsoft, and Nintendo use. The wizard branches; only categories that aren't trivially "no" are listed below. Default answer for everything not listed: **No**.

Target rating: **Everyone** (PEGI 3 / ESRB Everyone / USK 0).

---

## Category selection (first question)
Pick **Reference, News, or Educational** → **Other**. Some accounts only see the older "non-game" path → pick that.

> **Why not Parenting:** the rating questionnaire's "Parenting" sub-category is for apps targeting kids (e.g. educational kid games). This app is *for adults raising kids*, which IARC files under Reference / Other.

- Profanity? **No**

## Controlled substances
- Alcohol, tobacco, drugs depicted or referenced? **No**

## Gambling
- Real money, simulated, or in-app gambling? **No**

## User-generated content
- Does the app allow users to interact with each other (chat, share content)? **Yes — limited.**
  - **Direct interaction:** No public posts. Two adult users (household members) share data privately within the household.
  - **User-to-user content sharing:** Yes — within a household, users see each other's logged feedings and notes.
  - **Shares user location?** **No**
  - **Shares user info publicly?** **No**

> When asked: "Are user interactions moderated?" → **No moderation needed; visibility is gated to household members the user explicitly invites. No public timeline, no comments, no DMs.**

## Misc
- Crypto, encouraging illegal activity, hatred, sensitive social issues? **No**
- Personalised digital purchases? **No**

## Free-text "any other content reviewers should know"
*Optional, but recommended — it short-circuits clarification emails.*

```
Baby Food is a homemade-baby-food planning app for parents. The only "user-generated content" is private household data (feeding logs, recipes, photos of food, growth measurements) shared between two co-parents who explicitly invite each other via a one-time code. There is no public posting, no comments, no chat between strangers, and no advertising. The app is rated for everyone but its actual audience is adult parents and caregivers.
```
